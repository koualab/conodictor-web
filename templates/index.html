ConoDictor

ConoDictor is a tool for prediction of conopeptides superfamilies. It support large transcriptome as input.
We use Position Specific Scoring Matrix and Profile Hiden Markov Models to predict conopeptides superfamily.
The output file is a parsable tab-delimited file. Along with it we output a donut graph showing the distribution of superfamilies for the input file.
